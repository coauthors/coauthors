# Coauthors VSCode 익스텐션

GitHub username을 입력하고 커밋 메시지를 만들 수 있는 간단한 VSCode 확장 프로그램을 제공합니다.

## 사용 방법

<Steps>

### [VSCode 마켓플레이스](https://marketplace.visualstudio.com/items?itemName=coauthors.coauthors-vscode)에서 확장 프로그램을 설치합니다.

### Source Control의 아이콘 클릭 후 username을 입력합니다. 그게 전부입니다 🎉

확장 프로그램은 공동 작성자가 추가된 커밋 메시지를 자동으로 생성합니다.

![Example](/img/vscode-example.gif)

</Steps>
