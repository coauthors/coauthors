# Coauthors GitHub 앱

Coauthors GitHub 앱은 기여자가 댓글만 남긴 경우에도 자신의 커밋에 공동저자로 쉽게 추가할 수 있도록 공동저자추가 문법과 일치하는 문자열을 자동으로 제공합니다. 이는 GitHub 이슈와 풀 리퀘스트에서 작동합니다.

## 사용 방법

<Steps>

### [Coauthors GitHub 앱](https://github.com/apps/coauthors) 설치. 그게 전부입니다 🎉

댓글 작성자를 공동저자로 추가하기 위한 문자열을 쉽게 가져오세요. GitHub Pull Requests & Issues에서 공동저자추가 문자열을 직접 가져오세요. 그런 다음 커밋 메시지에 복사하여 붙여넣으세요.

</Steps>

![Example](/img/github-app-example.png)

많은 아이디어가 커밋뿐만 아니라 댓글을 통해서도 공유됩니다. 그러나 댓글만 남긴 아이디어 기여자를 공동저자로 추가하지 않으면 그들은 댓글로만 남게 되고 GitHub 저장소의 기여자 섹션에 포함되지 않습니다.

하지만 이 Coauthors GitHub 앱을 통해 커밋이 아닌 댓글을 통해서만 아이디어를 제공한 사람들이 기여자가 될 수 있으며, 이는 그들이 아이디어를 기여하도록 동기를 부여하고 저장소 내에서 더 많은 소통으로 이어질 수 있습니다.
